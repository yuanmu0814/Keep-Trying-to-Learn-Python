# 字典中元素的获取
## 方法：
  - `[]`: `scores['张三']`
  - `get()`方法： `scores.get('张三')`

```Python
print('-----第一种方式：使用[]-----')
scores = {'张三': 100, '李四': 98, '王五': 45}
print(scores['张三'])
print('-----第二种方式：使用gets()-----')
print(scores.get('张三'))
print(scores.get('陈六'))
print(scores.get('麻七', 99))
```

## `[]`取值与使用`get()`取值的区别
   - `[]`如果字典中不存在指定的`key`，抛出`KeyError`异常
   - `get()`方法取值，如果字典中不存在指定的`key`，并不会抛出`KeyError`，而是返回`None`，可以通过参数设置默认的`value`，以便指定的`key`不存在时返回
